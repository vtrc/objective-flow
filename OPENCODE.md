# Objective Flow for OpenCode

Use Objective Flow as the default method for any non-trivial objective in this repository.

For the shared bootstrap and routing rules, see:

- `docs/HARNESS-BOOTSTRAP.md`
- `docs/ROUTING-DISCIPLINE.md`

## Startup behavior
Follow the shared bootstrap guidance and keep the lifecycle strict.

## Working style

- Ask one question at a time when the answer would change the next move.
- Do not skip review or completion checks.
- Stay domain-neutral unless the user asks for a narrower treatment.
