# Problem Framing

Turn vague intent into a workable objective.

## Hard Gate

Do not advise, choose, plan, or act until the objective and success condition are clear enough for the next move.

## Process

1. Restate the current situation in one sentence.
2. Separate facts, assumptions, unknowns, preferences, and constraints.
3. Identify the underlying objective and the decision or action required.
4. Define an observable success condition.
5. Ask one blocking question when a missing answer would materially change the next move.
6. Choose the next move: answer, explore options, shape the approach, plan, act, diagnose, or review.

Use this compact format when helpful:

```markdown
Objective: ...
Decision or action: ...
Constraints: ...
Unknowns: ...
Success condition: ...
Next move: ...
```

Do not optimize a proposed solution before confirming it addresses the objective. Ask only questions that materially change the next move.
